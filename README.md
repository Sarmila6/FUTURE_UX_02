AuraDerm – Mobile Appointment Booking App
 Project Overview

AuraDerm Booking App is a mobile UI/UX design created to simplify appointment scheduling for a dermatology clinic. The focus of this project was to design a frictionless, user-friendly booking experience while maintaining warmth and trust in the brand.

 Problem Statement

Many clinic booking systems create friction due to:
Overly complex forms
Confusing time slot interfaces
Poor mobile usability
Lack of clear booking confirmation
This project aims to reduce cognitive load and streamline the entire booking journey.

 Target Users

First-time clinic visitors
Returning patients
Working professionals seeking fast booking
Users booking dermatology consultations

 User Flow

Onboarding → Login → Service Selection → Specialist Selection → Date → Time Slot → Review Booking → Confirmation
The flow was designed to minimize steps and reduce decision fatigue.

 UX Decisions & Real-World Considerations

OTP-based login reduces password friction
“Any Available Specialist” speeds up booking
Past dates are disabled to prevent invalid selections
Unavailable time slots are visually differentiated
Fixed bottom CTA improves thumb accessibility
Clear confirmation screen reduces booking anxiety

 Design System

Soft neutral background for warmth
Muted green accents to maintain medical trust
Rounded components for approachability
Large tap targets for mobile usability
